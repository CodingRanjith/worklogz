# Admin CRM Documentation

## Overview
The CRM (Customer Relationship Management) module provides three specialized pipelines for managing different types of business leads and customer relationships: Course CRM, Internship CRM, and IT Projects CRM. Each pipeline is tailored to specific business needs.

## Purpose
The CRM module enables administrators to:
- Manage customer leads and relationships
- Track lead progression through sales pipelines
- Monitor conversion rates
- Generate CRM reports
- Optimize sales processes

## Key Features

### CRM Pipeline Management
- **Pipeline Stages**: Manage stages in each pipeline
- **Stage Progression**: Move leads through stages
- **Stage Customization**: Customize pipeline stages
- **Stage Analytics**: Analyze performance by stage

### Lead Management
- **Lead Creation**: Create new leads
- **Lead Information**: Comprehensive lead information
- **Lead Assignment**: Assign leads to team members
- **Lead Tracking**: Track lead progress

### Lead Information Fields
- **Personal Information**: Name, email, phone, address
- **Lead Source**: Source of lead
- **Enquiry Date**: Date of initial enquiry
- **Lead Owner**: Assigned lead owner
- **Status**: Current lead status
- **Follow-up Information**: Follow-up dates and notes

## Course CRM

### Purpose
Manages leads for educational courses, training programs, and certification courses.

### Key Features
- **Course Selection**: Track selected courses
- **Program Types**: Training, Internship + Training, Certification, Workshop, Project Support, Placement Program
- **Training Modes**: Online, Offline, Hybrid
- **Batch Information**: Batch types and time slots
- **Payment Tracking**: Track course payments
- **Demo Management**: Track demo sessions
- **Trainer Assignment**: Assign trainers to leads

### Course-Specific Fields
- Course name and details
- Program type
- Training mode
- Batch type and time slot
- Course duration
- Course start date
- Payment status and methods
- Demo completion status
- Trainer information

---

## Internship CRM

### Purpose
Manages leads for internship programs, placement opportunities, and career development programs.

### Key Features
- **Internship Program Details**: Track internship program information
- **Candidate Information**: Comprehensive candidate details
- **Program Duration**: Track program duration
- **Placement Tracking**: Track placement status
- **Payment Management**: Manage internship payments
- **Progress Tracking**: Track candidate progress

### Internship-Specific Fields
- Internship program type
- Program duration
- Start date
- Placement status
- Payment information
- Progress tracking
- Skill assessment

---

## IT Projects CRM

### Purpose
Manages leads for IT projects, software development, and technical services.

### Key Features
- **Project Requirements**: Track project requirements
- **Client Information**: Comprehensive client details
- **Project Stages**: Enquiry, Requirement Collected, Proposal Sent, Negotiation, Closed/Won, Closed/Lost, On Hold
- **Project Status**: Not Started, In Progress, Under Review, Testing, Completed, Delivered, Support
- **Payment Tracking**: Track project payments
- **Timeline Management**: Manage project timelines
- **Hosting Preferences**: Track hosting requirements

### IT Project-Specific Fields
- Client type (Individual, Startup, Small Business, Enterprise, Internal Project)
- Industry type
- Project requirements
- Project timeline
- Budget information
- Payment terms
- Hosting preferences
- Support types

## Common CRM Features

### Lead Stages
- **Customizable Stages**: Each CRM has customizable stages
- **Stage Progression**: Move leads through stages
- **Stage History**: Track stage changes
- **Stage Analytics**: Analyze performance by stage

### Follow-up Management
- **Follow-up Scheduling**: Schedule follow-ups
- **Follow-up Notes**: Add follow-up notes
- **Follow-up History**: Track follow-up history
- **Follow-up Reminders**: Set follow-up reminders

### Activity Tracking
- **Activity Log**: Track all activities
- **Activity Types**: Calls, emails, meetings, etc.
- **Activity History**: Complete activity history
- **Activity Analytics**: Analyze activities

### Communication
- **Communication History**: Track all communications
- **Communication Types**: Call, WhatsApp, Email, In-person, Video Call
- **Last Contacted Date**: Track last contact
- **Communication Notes**: Add communication notes

### Payment Management
- **Payment Tracking**: Track payments
- **Payment Status**: Paid, Partial, Pending, Not Paid
- **Payment Methods**: Cash, UPI, Bank Transfer, Card, EMI
- **Payment History**: Complete payment history

### Reports and Analytics
- **Lead Reports**: Generate lead reports
- **Conversion Reports**: Track conversion rates
- **Pipeline Reports**: Analyze pipeline performance
- **Performance Analytics**: Analyze CRM performance

### Search and Filter
- **Search Leads**: Search by name, email, phone
- **Filter Options**: Filter by:
  - Stage
  - Status
  - Lead owner
  - Date range
  - Source
- **Advanced Search**: Advanced search options
